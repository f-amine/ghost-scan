Moroccan ID OCR App automatically associates each new API key with a unique project environment to ensure isolated data handling and enhanced security. You also have the option to integrate the API key into an existing project environment. Please note, all projects and API usage adhere strictly to our Privacy Policy and Terms of Service.

To utilize the Moroccan ID OCR App API, start by obtaining your unique API key. Once you have your key, you can make a POST request to the endpoint /ghost-scan/ocr/.
Ensure your request includes the following data format: {{ '{' }} 'image': [your_image_file], 'api_key': [your_api_key] {{ '}' }}.
This format is essential for the API to correctly process your request and perform the OCR on the provided Moroccan ID card image.
